## A Paper Airplanes

很简单的题，就考虑每个人需要用几个纸片，然后再算一下一共需要几包就行了。

## B Liebig's Barrels

先排序一下，用 $a_n-a_1$ 判断一下是否有合法操作。
然后，就贪心的从后往前选取 $n$ 个合法下界就行了。

## C Fair

$K \le 100$，所以我们可以跑 $k$ 次最短路，得到 $k$ 种物品每种物品到达这个点的最小花费是多少，然后每个点取前 $s$ 个花费就行了。

## D Easy Problem

DP题，我们考虑维护到这个点的最长前缀为l的代价为多少，取一下长度为 $0,1,2,3$ 的代价中的最小代价就好了。
这个题其实还有矩阵形式，相比较起来这个题挺简单。

## E Yet Another Substring Reverse

考虑到字母只有 $20$，而且问你一次翻转拼接，长度最长且字母不相同的字串，考虑状压 DP。
首先，对每一个位置预处理出往后衍生的合法串，记录到DP数组中。
然后对DP数组维护出当前状态子集下最长的串是多少。
最后，枚举两个不相交的集合合并就好了。

```c++
#include<bits/stdc++.h>
#define N 100005
typedef long long LL;
LL ans,a[N];
int n;
LL num[4],F[N][5];
char s[N];

  scanf("%d",&n);

  for (int i=1;i<=n;i++)scanf("%lld",&a[i]);

    for (int j=0;j<=3;j++)F[i][j]=F[i-1][j];
    if (s[i]=='h'){
      F[i][0]=F[i-1][0]+a[i];
      F[i][1]=std::min(F[i-1][1],F[i-1][0]);
    }else  if (s[i]=='a'){
      F[i][1]=F[i-1][1]+a[i];
      F[i][2]=std::min(F[i-1][2],F[i-1][1]);
    }else if (s[i]=='r'){
      F[i][2]=F[i-1][2]+a[i];
      F[i][3]=std::min(F[i-1][2],F[i-1][3]);
    }else if (s[i]=='d'){
      F[i][3]=F[i-1][3]+a[i];
    }
  }
  ans=F[n][0];
  ans=std::min(F[n][1],ans);
  ans=std::min(F[n][2],ans);
  ans=std::min(F[n][3],ans);
  printf("%lld",ans);
  return 0;
}
```

```c++
#include<bits/stdc++.h>
int F[(1<<20)+5];
int sta,now;
char s[1000005];
std::map<char,int>mp;
int count(int x){
  int num=0;
  while (x){
    if(x%2)++num;
    x/=2;
  }
  return num;
}
int main(){
  int num=0;
  scanf("%s",s+1);
  int len=strlen(s+1);
  for (int i=1;i<=len;i++){
    int l=i;
    int now=0;
    while (l<=i+19&&l<=len){
      if (!mp[s[l]])mp[s[l]]=++num;
      int x=mp[s[l]];
      x=1<<(x-1);
      if (now&x)break;
      else {
        now^=x;
        F[now]=count(now);
      //  printf("%d\n",F[now]);
        l++;
      }
    }
  }
  int sta=1<<num;
  sta--;
  for (int i=1;i<=sta;i++){
    for (int j=1;j<=num;j++)
      if (i&(1<<(j-1)))F[i]=std::max(F[i],F[i^(1<<(j-1))]);
  }
  int ans=0;
  for (int i=1;i<=sta;i++){
    ans=std::max(ans,F[sta-i]+F[i]);
  }
  printf("%d\n",ans);
  return 0;
}
```
